XML Parser

The GNUstep XML parser. This is an Objective-C wrapper for libxml. It provides for both event driven parsing, and walking a tree of autoimatically parsed xml elements.

The classes

  • GSXMLNamespace
  • GSXMLNode
  • GSXMLAttribute
  • GSXMLDocument
  • GSXMLParser
  • GSSAXHandler
  • GSHTMLParser - HTML specific subclass of GSXMLParser. No new API intorduced.
  • GSHTMLSAXHandler - HTML specific subclass of GSXMLSAXHandler. No new API intorduced.
